This is a program that generates truth tables. Enter your formula in the top bar and hit compile. The symbols are: 

                        ~    unary not
                        &    logical and
                        |    logical or
                        ->   implication
                        <->  equality
It will only accept well-formed-formulas.

An example formula looks something like (P & ~(Q | R))

Source is available here. A downloadable and runnable .jar file is available here

Please note: If you use n letter variables, you are generating 2^n lines. Don't expect this program to perform well if you end up using all the letters in the alphabet.
